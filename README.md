# ===================== OUTPUT EXPLANATIONS =====================

# Trend Plots (Line Charts):
# These plots show how mental health disorders (especially depression)
# change over time for each country from 1990–2017.
# By looking at the direction of the lines, we can identify whether
# disorder rates are increasing, decreasing, or staying relatively stable.
# This helps us understand long-term global and country-specific trends.


# Correlation Heatmap:
# This visualization displays the relationships between key variables
# such as depression, HDI, GDP, and unemployment rate.
# Each value ranges from -1 to 1:
#   - Values close to 1 indicate a strong positive relationship
#   - Values close to -1 indicate a strong negative relationship
#   - Values near 0 indicate little to no relationship
# This helps identify which factors may be associated with higher or lower depression rates.


# Scatter Plot (HDI vs. Depression):
# Each point represents a country, comparing its average HDI with its
# average depression rate.
# This allows us to visually assess whether more developed countries
# tend to have higher or lower depression levels.
# It also helps highlight outliers—countries that do not follow the
# general pattern—indicating that other factors may be influencing results.


# Top/Bottom Country Comparisons:
# By calculating average depression rates, we identify the countries
# with the highest and lowest levels of depression.
# This helps focus our analysis on extreme cases, making it easier
# to compare patterns and investigate possible causes.


# Forecast (Regression Output):
# Linear regression is used to model past trends and predict future
# disorder prevalence.
# The predicted values (often shown as dashed lines) estimate how
# depression and other disorders may change in future years.
# This provides insight into potential long-term patterns, although
# predictions are based on past data and may not account for new events.


# Overall Takeaway:
# These outputs work together to provide a comprehensive understanding
# of global mental health trends, relationships between variables,
# and possible future outcomes.
